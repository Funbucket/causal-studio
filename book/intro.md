# Causal Studio

인과추론 교육 영상과 실행 가능한 코드북을 제공하는 플랫폼입니다.

## 학습 방법

1. **영상 시청**: 각 챕터의 3Blue1Brown 스타일 애니메이션으로 개념을 시각적으로 이해
2. **코드 실행**: Binder 또는 Colab에서 직접 코드를 실행하며 실습
3. **심화 학습**: 노트북의 상세 코드와 설명으로 깊이 있는 이해
